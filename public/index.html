Plomo API Documentation

Use this page to read API contracts and test requests directly from browser.

Base URL

Local: http://localhost:3000

Endpoints

POST /api/auth/signin

Create parent account.

{
  "username": "raghu_parent",
  "childType": "boy",
  "parentName": "Raghu",
  "mobileNumber": "9876543210",
  "dateOfBirth": "05/04/2018",
  "password": "secret123"
}
POST /api/auth/login

Parent login with username or mobile number.

{
  "identifier": "raghu_parent",
  "password": "secret123"
}
POST /api/auth/kid-login

Kids login without password.

{
  "kidName": "Aarav",
  "dateOfBirth": "05/04/2018"
}
POST /api/kids

Create kid profile. Requires parent Bearer token.

{
  "kidName": "Aarav",
  "childType": "boy",
  "dateOfBirth": "05/04/2018"
}
POST /api/tasks

Create task. Requires parent Bearer token.

{
  "parentName": "Raghu",
  "mobileNumber": "9876543210",
  "kidName": "Aarav",
  "taskType": "study",
  "taskTitle": "Math homework",
  "taskDescription": "Complete chapter 2 exercises",
  "taskPoints": 10,
  "taskStatus": "new"
}
POST /api/tasks/list

Get tasks list for one kid. Requires parent Bearer token.

{
  "parentName": "Raghu",
  "mobileNumber": "9876543210",
  "kidName": "Aarav"
}
