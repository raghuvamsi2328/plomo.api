Quick Flow

Follow this exact order for a clean test run in browser or Postman.

Step 1

POST/api/auth/signin

Create parent account.

Step 2

POST/api/auth/login

Copy data.token from response and use as Bearer token.

Step 3

POST/api/kids

Create kid profile using parent token.

Step 4

POST/api/tasks

Create tasks for the kid.

Step 5

POST/api/tasks/list

Fetch task list for the kid.

Kids Login
